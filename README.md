# planet-weight-calculator
A Python OOP project that calculates your weight on each planet in the Solar System.
My first project in Python, also with application of OOP
Its just a simple app where you can calculate your weight (in earth) on each planet in solar system.
It's not a fast track coding, it takes time and understanding on how to apply OOP in dictionary, how to properly call them and discover some python stuff that I didn't know (for example, :.2f in f string)
I hope you enjoy my first project and feel free if you have any suggestion or feedback to improve my code/app. thanks!
